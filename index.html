FireEdit is a js editor based on ACE and narcissus. It runs as a plugin in firefox 7.x. user ctrl-shift-e to activate the editor, this will open a new tab with the editor. Ignore the url displayed in the editor, it points to the addon url on the local machine. Currently works with Firefox 7. Feature List :
  1. Work with local files (completed)
  2. Method window
  3. Refactoring
    1. Method rename
    2. Variable rename
    3. Extract Methods
  4. Key customization support
  5. Custom save actions
  6. Debugging
